frc2::CommandPtr CoralSubsystem::GoToAngle ( units::degree_t targetAngle)

Set the desired setpoint angle for the coral scoring mechanism. This function only changes the saved setpoint. The Periodic function is responsible for tracking the setpoint and holding the motor at the angle.

Parameters
targetAngleThe desired angle of the mechanism in degrees
